# 더에이아이랩(The AI Lab) 홈페이지 개발 PRD

## 프로젝트 개요
더에이아이랩은 누구나 쉽게 배우고, 일상과 업무에서 실질적으로 활용할 수 있도록 보편적 인공지능 시대를 지향하는 에듀테크 스타트업입니다. 
본 프로젝트는 더에이아이랩의 비전, 히스토리, 기업문화, 제품, 뉴스, 파트너, 채용 정보를 소개하는 공식 홈페이지를 구축하는 것을 목표로 합니다.

## 기술 스택
- **프레임워크**: React 19.2 + TypeScript
- **빌드 도구**: Vite 7.2
- **스타일링**: Tailwind CSS v4
- **UI 컴포넌트**: shadcn/ui (new-york style)
- **아이콘**: lucide-react
- **라우팅**: React Router v6
- **상태관리**: Context API (필요시)
- **애니메이션**: Framer Motion (선택적)

### 4. Culture 페이지
**메인 배너:**
- "일이 즐겁도록"
- 타이틀: "OUR Culture "
- 소개: 현대인들은 하루의 1/3 이상을 회사에서 보내기에 그 시간이 즐거워야 삶이 좀 더 행복하고 윤택해질 수 있습니다.
 회사에서 일하는 시간이 즐겁기 위해서는 건강하고 행복한 기업 문화가 갖추어져야 합니다.

**우리의 기업문화 (7가지 원칙):**
1. 스타트업 마인드를 우리의 DNA에 새기고 기업의 미션과 비전을 실현하기 위해 노력한다
2. 한계를 뛰어넘는 목표를 설정하고 이를 통해 함께 성장한다
3. 자신의 역할에 한계를 짓지 않으며 유연한 팀을 운영한다
4. 모든 업무 과정은 투명하게 공유한다
5. 누구나 실패할 수 있다. 다만 성공적인 실패가 되어야 한다
6. 직급, 나이에 관계없이 상호 간 존칭을 사용하고 존중하고 배려한다
7. 우리는 지구의 보편적 삶과 교육을 향상시키기 위해 노력한다

**업무방식 (6가지):**
1. 한계를 뛰어넘는 목표와 퍼포먼스
2. 투명한 공유
3. 유연한 팀과 책임감있는 리더십
4. 자기성찰과 피드백
5. 적극적인 커뮤니케이션
6. Open Door Policy

각 항목은 아이콘/이미지와 함께 상세 설명 포함

**Culture :**
- 존중과 이해, 상호작용과 소통에 대한 7가지 가치 설명
- 우리는 함께 이 정글을 헤쳐나갈 동료입니다
- 서로를 인정하고 칭찬하고 돕습니다
- 소통을 위한 용기
- 이타심과 선한 영향력
- 나 먼저, 함께 역지사지
- 생각하는 동료
- 편견을 가지지 않고 차별하지 않습니다
- 틀린 것이 아니라 다른 것 입니다
