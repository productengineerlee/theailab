## 🚀 배포 방법

### **빠른 시작 (3단계)**

#### **1단계: Supabase 설정** (10분)
```
1. https://supabase.com 접속
2. 새 프로젝트 생성 (Region: Seoul)
3. SQL Editor에서 실행: sql/00-complete-setup.sql
4. Settings > API에서 URL & Key 복사
```

#### **2단계: Vercel 배포** (5분)
```
1. https://vercel.com 접속
2. Import Git Repository
3. Framework: Vite 선택
4. 환경 변수 입력:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
5. Deploy 버튼 클릭
```

#### **3단계: 확인** (5분)
```
✓ 배포 URL 접속
✓ 모든 페이지 확인
✓ 한/영 번역 확인
✓ Admin 로그인 테스트
```
